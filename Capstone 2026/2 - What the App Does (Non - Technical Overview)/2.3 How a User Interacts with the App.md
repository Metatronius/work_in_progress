1. **Open the app** → land on the main task list screen

2. **Tap the blue "Add Task" button** to create a new task

3. **Fill in the title** (required), and optionally a due date, priority, notes, and reminder flag

4. **Tap Save** → the new task appears at the top of the list

5. **Tap a checkbox** to mark a task complete or incomplete

6. **Tap a task title** to see full details on a separate screen

7. **Long-press a title** → confirm deletion → task is removed

8. **Type in the search bar** → list filters instantly

9. **Tap a task to edit** → modify any field and save changes
