The main entry point and primary user interface. Displays all tasks in a scrollable list and provides search functionality.

  
**Responsibilities:**

- Fetch and display all tasks from the ViewModel

- Observe `viewModel.allTasks` LiveData for real-time updates

- Implement a search bar that filters tasks by title

- Navigate to `AddTask` when the user taps the "Add Task" button

- Navigate to `TaskDetail` when a task is tapped

- Navigate to `EditTask` or handle delete when long-pressing a task

  

**Key Interactions:**

- Search bar uses a `TextWatcher` to listen for user input and filters `currentTasks` in-memory (no database queries)

- Each task item in the list is clickable to view details

- Tapping the floating action button (FAB) launches the `AddTask` screen with `startActivityForResult()`

- Long-press options trigger an `AlertDialog` for edit/delete confirmation

  

**Layout:**

- `activity_main.xml` uses a `LinearLayout` with a search bar at the top and a scrollable task list below

- Each list item shows: task title, priority indicator (color-coded), due date (if set), and a checkbox for completion
