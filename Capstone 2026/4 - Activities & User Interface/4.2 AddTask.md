A dedicated screen for creating new tasks. Collects all task metadata and validates before saving.

  
**Responsibilities:**

- Display a form with input fields for title, notes, priority, due date, and reminder flag

- Call `viewModel.addTask(TaskParams(...))` when the user taps "Save"

- Validate all inputs before submission

- Return to `MainActivity` on successful save

  

**Fields:**

- **Title** (EditText) – Required, max 30 characters

- **Notes** (EditText, multiline) – Optional, allows any length

- **Priority** (Spinner) – Dropdown: None, Low, Medium, High

- **Due Date** (DatePicker) – Optional; if provided, must be a valid date

- **Reminder** (CheckBox) – Toggle to enable/disable reminder flag

- **Buttons** – Save (validates and inserts) and Cancel (returns to MainActivity)

  

**Validation:**

- Title cannot be blank

- Title cannot exceed 30 characters

- Priority must be one of the enum values (0–3)

- Due date (if provided) must be in a valid format (validated by `DataUtil.validateDate()`)
