Allows users to modify an existing task. Pre-populates all fields with the current task data.

  
**Responsibilities:**

- Load the task by ID (passed via intent extra)

- Pre-populate all form fields with current values

- Call `viewModel.updateTask(...)` when the user taps "Save"

- Return to `MainActivity` or `TaskDetail` on successful update

  

**Behavior:**

- On activity creation, fetch the task from the ViewModel and populate the UI

- All validation rules from `AddTask` apply here as well

- "Discard" button returns without saving
