A read-only view that displays all information for a single task in one place.

  
**Responsibilities:**

- Load and display the task by ID

- Show all fields: title, notes, priority, due date, reminder status, creation date, completion status

- Provide navigation options to edit or delete the task

- Return to `MainActivity` when done

  
**Display:**

- Title (large, prominent)

- Priority (badge or color-coded label)

- Due date (if set)

- Notes (full text area)

- Reminder status (icon or toggle, read-only)

- Creation date/time

- Progress/completion indicator

- Buttons: Edit, Delete, Back
