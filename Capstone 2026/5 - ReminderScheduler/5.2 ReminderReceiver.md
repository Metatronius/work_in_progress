
BroadcastReceiver that fires when alarm triggers and displays notification


**Responsibilities:**

- Receives broadcast from AlarmManager at scheduled time

- Creates NotificationChannel ("task_reminders") for task reminder notifications

- Builds and displays notification with task title

- Uses IMPORTANCE_HIGH for priority notifications



**Display:**

- Channel ID: "task_reminders"

- Title: "Task Due Soon"

- Content: Task title

- Icon: Android system alarm icon

- Auto-cancels when user taps it
