Defines SQL operations with Room annotations:

  
- `getAllTasks(): Flow<List<Task>>` – Emits full list whenever database changes

- `getTaskById(id: Int): Task?` – Returns single task or null

- `insertTask(task: Task)` – Inserts new task

- `updateTask(task: Task)` – Updates existing task

- `deleteTask(task: Task)` – Deletes task
