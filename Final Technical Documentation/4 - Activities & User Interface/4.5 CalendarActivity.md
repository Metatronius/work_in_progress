A calendar to manage all active and completed tasks at the time they are due.

**Responsibilities:**

- Displays an interactive calendar view

- Shows tasks filtered by selected date

- Allows toggle between daily and weekly task views

- Shows the task count per day visually



**Behavior:**

- Receives task list from MainActivity via intent

- User selects the date on the calendar to see tasks due that day

- Toggle button switches between daily and weekly views

- Bottom navigation allows quick return to MainActivity
