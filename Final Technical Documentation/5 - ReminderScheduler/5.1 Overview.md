```kotlin
object ReminderScheduler {

    fun schedule(context: Context, taskId: Int, taskTitle: String, dueDate: String)

    fun cancel(context: Context, taskId: Int)

}

```

**Responsibilities:**

- `schedule()` - Parses the due date and sets AlarmManager to trigger 1 day before at 9 am

- `cancel()` - Cancels pending reminder for a task

- Uses `PendingIntent` with `FLAG_UPDATE_CURRENT` and `FLAG_IMMUTABLE`

- Checks `canScheduleExactAlarms()` and falls back to `set()` if not available
