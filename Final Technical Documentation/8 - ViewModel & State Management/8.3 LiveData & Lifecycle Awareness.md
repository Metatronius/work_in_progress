Data exposed as LiveData for lifecycle awareness:

  
```kotlin

val allTasks: LiveData<List<Task>> = taskRepository.allTasks.asLiveData()

```


**Benefits:**

- Lifecycle-aware (pauses when Activity destroyed)

- Reactive (UI updates automatically)

- Thread-safe (main thread only)

- Survives rotation
