- **Task List**: Users log in and view all their tasks on a single scrollable home screen.

- **Create a Task**: Users fill out a form with a title, due date, priority level, notes, and an optional reminder flag.

- **Edit a Task**: Users can modify any task field (title, priority, due date, notes, reminder) by opening the edit screen.

- **Search**: A live search bar filters the task list by title as the user types.

- **Complete a Task**: Each task has a checkbox. Tapping it toggles the task between incomplete and complete.

- **Task Detail**: Tapping a task title opens a read-only detail screen showing all task fields.

- **Delete a Task**: Long-pressing a task title shows a confirmation dialog before permanently deleting it.

- **Offline-First**: All data is stored in a local SQLite database. No network access is required.

- **Automated Documentation**: A GitHub Actions workflow auto-generates KDoc comments for every undocumented Kotlin declaration on every push.
