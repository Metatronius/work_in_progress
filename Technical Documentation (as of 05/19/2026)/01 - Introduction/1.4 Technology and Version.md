**Other Technologies:**

- **Dependency Management:** The project manages dependencies and plugin versions centrally using a `libs.versions.toml` file.

- **Core Build Versions:** The application is built using Kotlin version 2.0.21 and Android Gradle Plugin (AGP) version 8.13.2. The Gradle wrapper is strictly configured to use Gradle distribution version 9.3.1.

- **Java Toolchain:** The JVM toolchain is configured for Java version 21. Toolchain resolution is mapped via the Foojay API for cross-platform compatibility across Linux, macOS, Windows, and FreeBSD.

- **Architecture Libraries:** Key Android dependencies include Room version 2.6.1 for SQLite persistence and Lifecycle version 2.8.7 for ViewModel and LiveData management. Database annotation processing is handled by KSP (Kotlin Symbol Processing).
