Each task in the database has these fields:

|           |          |                                               |              |
| --------- | -------- | --------------------------------------------- | ------------ |
| **Field** | **Type** | **Description**                               | **Default**  |
| id        | Int      | Unique ID (auto-generated primary key)        | Auto         |
| title     | String   | Short name of the task (1–30 chars, required) | —            |
| notes     | String   | Optional longer description                   | "" (empty)   |
| priority  | Int      | 0 = None, 1 = Low, 2 = Medium, 3 = High       | 0            |
| created   | String   | Timestamp when the task was created           | Current time |
| due       | String?  | Optional due date (null if not set)           | null         |
| remind    | Boolean  | Whether a reminder was requested              | false        |
| progress  | Int      | 0 = incomplete, 1 = complete                  | 0            |
| target    | Int      | Target completion value (always 1)            | 1            |
