- **Copyright Header Insertion:** The pipeline automatically scans all Kotlin files and inserts a standard copyright header if one is missing.

- **KDoc Stub Insertion:** A Python script (`insert_kdoc.py`) scans the codebase and inserts a `/ TODO: add documentation. */` placeholder above any undocumented functions, classes, interfaces, or objects.

- **AI-Powered Generation:** A secondary Python script (`generate_kdoc.py`) extracts the code surrounding the TODO stubs and sends it to the GitHub Models API. It utilizes the `gpt-4o-mini` model to generate concise, properly formatted KDoc comments (including `@param` and `@return` tags).

- **Auto-Commit:** Modified files are automatically committed and pushed back to the repository. The workflow is configured to skip runs triggered by its own bot commits to prevent infinite loops.
