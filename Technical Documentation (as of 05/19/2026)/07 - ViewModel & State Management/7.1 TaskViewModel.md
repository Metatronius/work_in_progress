**Exposed Data:**

- `allTasks: LiveData<List<Task>>` – Observable stream of all tasks

  
**Operations:**


- `addTask(newTask: TaskParams)`

  - Validates: title not blank and 0–30 chars

  - Validates: due date format if provided

  - Throws `IllegalArgumentException` if validation fails

  - Inserts task inside coroutine

-  `editTask(id, title, notes, priority, due, remind, progress, target)`

  - Updates existing task

- `deleteTask(task: Task)`

  - Deletes task inside coroutine 

- `deleteTaskById(id: Int)`

  - Looks up task by ID and deletes it

- `completeTask(task: Task)`

  - Toggles progress between 0 and 1
