Using Gradle with Kotlin DSL:

```gradle

android {

    namespace = "com.example.work_in_progress"

    compileSdk = 35

    minSdk = 31

    targetSdk = 35

}

```

  

Build commands:

```bash

./gradlew build              # Build APK

./gradlew test              # Run unit tests

./gradlew connectedAndroidTest  # Run instrumented tests

```
